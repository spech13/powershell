POWERSHELL

SET-LOCALUSER
Set-LocalUser -Name <user_name> -Description <description> - description changing of user named "user" to "description"

REMOVE-ITEM
Remove-Item <item> - remove item named "item"
Remove-Item <item> -Force - remove item named "item", contained other items

RENAME-LOCALUSER
Rename-LocalUser -Name <current_name> -NewName <new_name> - user name changing from "current_name" to "new_name"

GET-LOCALUSER
Get-LocalUser <user_name> - get object of local user with name "user_name"
Get-LocalUser -Name <user_name> - get object of local user with name "user_name"
Get-LocalUser -SID <user_sid> - get object of local user with sid "user_sid"

SELECT-OBJECT
<object> | Select-Object <properties_list> - getting list of properties specified in "list_properties" of object "objects"

GET-MEMBER
<object> | Get-Member - getting list of object properties

NET USER
net user - list of all local users
net user <user_name> - information about user with name "user_name"
net user <user_name> /add - user adding with name "user_name"
net user <user_name> /del - user deleting with name "user_name"

NET LOCALGROUP
net localgroup - list of all local groups
net localgroup <group_name> list users of local group with name "group_name"
net localgroup <group_name> /add - local group adding with name "group_name"
net localgroup <group_name> /del - local group deleting with name "group_name"
net localgroup <group_name> <user_name> /add - user adding with name "user_name" to local group with name "group_name"
net localgroup <group_name> <user_name> /del - user deleting with name "user_name" to local group with name "group_name"
